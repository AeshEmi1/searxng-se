.. _settings brand:

==========
``brand:``
==========

.. code:: yaml

   brand:
     issue_url: https://github.com/AeshEmi1/searxng-se/issues
     docs_url: https://aeshemi1.github.io/searxng-se/
     wiki_url: https://github.com/AeshEmi1/searxng-se/

``issue_url`` :
  If you host your own issue tracker change this URL.

``docs_url`` :
  If you host your own documentation change this URL.

``wiki_url`` :
  Link to your wiki (or ``false``)
